# Servo

The `servo` extension controls the angle of a servo or the rotation speed of a continuous servo

```blocks
modules.servo1.setAngle(90)
```

## Angle

Sets the angle in degrees.

```blocks
modules.servo1.setAngle(90)
```

## Continuous servos

Run at 100% throttle.

```blocks
modules.servo1.run(100)
```

Stop

```blocks
modules.servo1.stop()
```
